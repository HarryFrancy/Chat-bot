AI Chatbot with RAG

A chatbot that lets you chat with any PDF using Groq, LangChain, and Streamlit.

Features
- Upload any PDF and ask questions about it
- Persistent chat history
- Multiple personality modes (Teacher, Strict, Chill)
- Built with Groq LLM, LangChain, ChromaDB, Streamlit

How to Run
1. Install dependencies:
pip install streamlit groq langchain langchain-community chromadb sentence-transformers pypdf

2. Add your Groq API key in the code

3. Run:
streamlit run yakuza.py
